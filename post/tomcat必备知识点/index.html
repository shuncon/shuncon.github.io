tomcat必备知识点

@SHUNCON · Nov 17, 2018 · 1 min read

tomcat隐藏版本信息

1.进入tomcat的lib目录找到catalina.jar文件

unzip catalina.jar 得到两个文件夹： META-INF org

编辑org中的文件

vim org/apache/catalina/util/ServerInfo.properties

server.info=Win  IIS
server.number=0.0.0.0
server.built=Apr 3 2018 20:04:09 UTC
改成这样的

重新打包更新jar的文件内容

cd  /tomcat/lib
#jar uvf catalina.jar org/apache/catalina/util/ServerInfo.properties

增加tomcat使用内存量

 vim bin/catalina.sh 
 
 cygwin=false #在此行上面添加以下代码
 set JAVA_OPTS=-server -Xms512m -Xmx1024m -XX:MaxNewSize=512m -XX:MaxPermSize=256m

解析一下内存参数：

 -Xms512m 内存最小使用，-Xmx1024m内存最大使用

tomcat优化参数：

#第一部分：
<?xml version='1.0' encoding='utf-8'?>
<Server port="9090" shutdown="SHUTDOWN">  #此处自定义为9090
<Connector port="9091" protocol="HTTP/1.1"   #此处自定义为9091
                maxThreads="1000"
                minSpareThreads="200"
                maxSpareThreads="800"
                acceptCount="1100"
               connectionTimeout="60000"
               redirectPort="8402"
                executor="tomcatThreadPool"
                debug="0"
                enableLookups="false"
                disableUploadTimeout="true"
                URIEncoding="UTF-8"
                useBodyEncodingForURI="true"
#第二部分：
指定网站首页JSP、JS等文件存放位置
 <Context docBase="/data/www/shunconf/webroot" path="" reloadable="false"  debug="0">
       </Context>  定义网站代码存放的路径
