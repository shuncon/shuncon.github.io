ansible简单使用

@SHUNCON · Nov 10, 2018 · 1 min read

环境

ansible 服务端 和两台客户端必须建立ssh免密认证：

本次安装采用：yum -y install ansible
ansible 服务端操作：
ssh-keygen
ssh-copy-id -i  root@10.0.18.101
建立免密认证后 ssh root@10.0.18.101验证一下是否成功

编辑ansible host文件

添加两个在host文件中，该hosts文件属于ansible的管理清单

vim /etc/ansible/hosts

SSH端口不是22端口则在该hosts文件中使用以下方法：

ansible_ssh_host=10.0.18.11   #主机iP
ansible_ssh_port=3389         #SSH端口
ansible_ssh_user=root         #ssh所远程的系统用户
ansible_ssh_pass=shunconf     #用户密码

简单使用

#ansible all -a '/bin/free -m' 查看主机内存使用和硬盘空间，通过系统的命令实现！

#ansible all -m ping -u root
#-u是指定用户，可以不加的。ping是ansible中的一个模块 -m参数则表示调用该模块。 all参数代表全部。ansible 10.0.18.101 -m ping则ping指定IP

主机可以ping通：

获取文件到ansible主机

#ansible all -m fetch -a 'src=/etc/resolv.conf dest=/opt' 使用fetch模块实现，-a选项则传递模块使用的参数：src与dest
