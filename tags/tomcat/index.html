zabbix监控tomcat

@SHUNCON · Dec 10, 2018 · 1 min read

版本 zabbix4.0 前提条件 编译 编译zabbix server时需要提供java支持，即添加--enable-java该选项。 系统也要安装有jdk版本 yum yum -y install zabbix-java-gateway 开始监控 修改zabbix_server.conf vim/etc/zabbix/zabbix_server.conf JavaGateway=10.0.18.118 //java网关地址，即server端ip地址 JavaGatewayPort=10052 //网关端口，取消注释开启即可 StartPollers=50 StartPingers=100 StartJavaPollers=5 开启java配置已经完成。 vim /etc/zabbix/zabbix_java_gateway.conf LISTEN_IP="0.0.0.0" #监听的IP LISTEN_PORT=10052 #监听的端口 PID_FILE="/var/run/zabbix/zabbix_java.pid" #pid保存路径 START_POLLERS=5 #这是默认启动的监听jmx的线程数，可以根据监控的jmx数量修改成一致的，这样一次就可采集完成所有的jmx状况 TIMEOUT=20 systemctl restart zabbix-server systemctl start zabbix-java-gatewa systemctl enable zabbix-java-gatewa 重启，即可 修改客户端中的tomcat文件 vim bin/catalina.sh JAVA_OPTS="$JAVA_OPTS -Djava.net.preferIPv4Stack=true -Djava.net.preferIPv4Addresses=true " //web端使用ipv4协议，默认使用ipv6 CATALINA_OPTS="$CATALINA_OPTS -Dcom.sun.management.jmxremote -Dcom.sun.management.jmxremote.authenticate=false -Dcom.sun.management.jmxremote.port=12345 -Dcom.sun.management.jmxremote.ssl=false -Djava.

tomcat必备知识点

@SHUNCON · Nov 17, 2018 · 1 min read

tomcat隐藏版本信息 1.进入tomcat的lib目录找到catalina.jar文件 unzip catalina.jar 得到两个文件夹： META-INF org 编辑org中的文件 vim org/apache/catalina/util/ServerInfo.properties server.info=Win IIS server.number=0.0.0.0 server.built=Apr 3 2018 20:04:09 UTC 改成这样的 重新打包更新jar的文件内容 cd /tomcat/lib #jar uvf catalina.jar org/apache/catalina/util/ServerInfo.properties 增加tomcat使用内存量 vim bin/catalina.sh cygwin=false #在此行上面添加以下代码 set JAVA_OPTS=-server -Xms512m -Xmx1024m -XX:MaxNewSize=512m -XX:MaxPermSize=256m 解析一下内存参数： -Xms512m 内存最小使用，-Xmx1024m内存最大使用 tomcat优化参数： #第一部分： <?xml version='1.0' encoding='utf-8'?> <Server port="9090" shutdown="SHUTDOWN"> #此处自定义为9090 <Connector port="9091" protocol="HTTP/1.1" #此处自定义为9091 maxThreads="1000" minSpareThreads="200" maxSpareThreads="800" acceptCount="1100" connectionTimeout="60000" redirectPort="8402" executor="tomcatThreadPool" debug="0" enableLookups="false" disableUploadTimeout="true" URIEncoding="UTF-8" useBodyEncodingForURI="true" #第二部分： 指定网站首页JSP、JS等文件存放位置 <Context docBase="/data/www/shunconf/webroot" path="" reloadable="false" debug="0"> </Context> 定义网站代码存放的路径
