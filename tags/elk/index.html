ELK集群 部署

@SHUNCON · Jan 1, 0001 · 2 min read

ELK集群 环境 es-node1 10.0.18.118 主节点 es-node2 10.0.18.117 从节点 用户权限说明： 除了elasticsearch不能用root运行，其他程序均可使用root安装 es-node1 操作 安装 #yum -y install jre #wget https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-6.6.1.tar.gz #tar xf elasticsearch-6.6.1.tar.gz -C /data/server/es/ 创建相关目录 #mkdir -p /data/server/es/elasticsearch-6.6.1/logs #mkdir /data/es/data -p #数据存储目录 编辑配置文件 #cat elasticsearch.yml|grep -v ‘#’ cluster.name: ELK node.name: es-node1 #节点唯一名称，不可和其他节点重复 node.master: true node.data: true node.ingest: true node.attr.rack: r1 path.data: /data/es/data path.
